# Je ne sais pourquoi
Je ne sais pourquoi
Mon esprit amer
D'une aile inquiète et folle vole sur la mer.
Tout ce qui m'est cher,
D'une aile d'effroi
Mon amour le couve au ras des flots. Pourquoi, pourquoi ?
